# Hotel Seren – Hospitality Management System 🏨✨
Hotel Seren is a comprehensive web-based platform that simulates real-world hotel operations coordination between guests and administrators to ensure seamless hospitality experiences and efficient room management.
---

## 🚀 Project Overview  
Hotel Seren streamlines the complete hotel management lifecycle by:
- Enabling guests to book rooms and access amenity services (book.html, guest_amenities.html)
- Empowering admins to manage room availability and track guest records (admin_rooms.html, admin_records.html)
- Providing role-based logins and secure access (guest-login, admin-login)

**Built With:**
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Python Flask
- Database: PostgreSQL (Neon)
- UI: Elegant, responsive interface with glassmorphism design
---

## 🔑 Key Features

### 🧑‍💼 Guest Portal
- Book rooms with real-time availability checking
- Order food, beverages, and services through room amenities
- View detailed billing with GST calculations and make payments
- Secure guest login with auto-generated guest ID tracking

### 🏢 Admin Dashboard
- Monitor and update room status (Available/Occupied/Maintenance)
- Access complete guest records and booking history
- Track revenue and occupancy statistics in real-time
- Full control over room inventory and pricing

### 🗄️ Database & Booking Management
- PostgreSQL-powered schema with optimized tables for:
  - Guests & Bookings
  - Rooms & Inventory
  - Amenity Orders & Billing
- Booking status: Checked-in, Checked-out, or Active
- Automated guest ID generation and room allocation

### 🎯 Service & Support System
- Integrated room service ordering with menu management
- Automated billing with tax calculations
- Real-time room availability updates
- Comprehensive reporting and analytics for admin insights
